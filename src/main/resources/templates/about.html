About Scissy

What is Scissy ?

Scissy is a personal project of a simple weather interface for the Channel Islands, using the data from OpenWeatherMap.

Why this name ?

The name comes from the legendary forest of Scissy. A forest that, following its myth, spread itself on the Mont-Saint-Michel's bay before disappearing in one night, due to a tsunami.

How does it work?

Scissy is a Server Side Rendering Web App with independent location, weather and air condition's databases actualized automatically by multi-threading.
